# 3D Digital Grillz Pipeline

A prototype website showcasing my Bachelor's Thesis on a digital pipeline for custom 3D dental grillz.

## About

This React web application demonstrates the multidisciplinary approach to designing dental grillz, bridging traditional craftsmanship with digital innovation. The project explores the complete workflow from digital scanning to final product visualization.

## Tech Stack

- **React** - UI framework
- **Vite** - Build tool
- **Three.js** - 3D rendering
- **React Router** - Navigation
- **GitHub Pages** - Deployment

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
